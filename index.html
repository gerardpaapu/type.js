Type.js

Type.js provide the class Type, which allows programmers to treat the variety of type questions relevant to coding in javascript as values rather that abstract ideas.

Types can be created from constructors

Type.from(Array).check([]) // true!

Type.from(String).check("") // true!

You may also use Type.check as a short-cut as in

Type.check(Array, []) // true!

Some types are more specific than others

Type.Array.moreSpecificThan(Type.Object) // true! Type.Integer.moreSpecificThan(Type.Number) // true!

Makes javascript types less scary

Checking for null, the arguments object, or NaN can be a little weird.

In Type.js they are represented by Type.Null, Type.Arguments, and Type.NaN.
